Preprocessing¶

This module provides various functions used to read/write and generate the data structures used for Path Learn

preprocessing.add_Ts(T0, T1)¶

Merges two path dictionaries.

Parameters

  • T0 – A path dictionary.

  • T1 – A path dictionary.

Returns

A merged path dictionary.

preprocessing.add_neg_samples(G, pos_pairs, samp_size, steps=3)¶

Performs negative sampling.

Parameters

  • G – NetworkX graph.

  • pos_pairs – A collection of pairs with edges.

  • samp_size – Negative samples per existing edge.

  • steps – Max length of random walk.

Returns

A list of positive and negative node pairs and their labels.

preprocessing.add_paths(G, pair_set, steps, rev_rel, T)¶

Adds new paths to path dictionary T.

Parameters

  • G – Networkx graph.

  • pair_set – A collection of pairs.

  • steps – Maximum path length.

  • rev_rel – Reverse relation type, for directed graphs.

  • T – A path dictionary T.

Returns

A path dictionary T that includes paths for pairs in pair_set.

preprocessing.find_paths_between(G, start, end, length)¶

Parameters

  • G – NetworkX graph.

  • start – Start node.

  • end – End node.

  • length – Maximum path length.

Returns

A set with all paths up to length fron start to ends.
